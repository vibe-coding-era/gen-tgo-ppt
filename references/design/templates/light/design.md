# Light Template Design

Load this file when the user chooses light template or no template is specified for normal business presentations.

## Asset

- PPTX: `assets/design/templates/light/tgo-gtlc-light.pptx`
- Original source: `GTLC-PPT模版-浅色.pptx`
- Visual role: pale blue-gray content pages, dark blue cover/back cover, GTLC brand frame.

## Layouts

- `封面`: opening cover.
- `1_大标题`: centered section divider.
- `大标题`: section page with body.
- `自我介绍`: speaker/self introduction.
- `目录`: agenda/table of contents.
- `内容1-大字标题`: content page with large title.
- `内容2-小字标题`: default content page.
- `空内容`: blank branded page.
- `封底`: closing/thanks.

## Key Positions

- Cover title: `x=1.800 y=2.544 w=18.684 h=3.103`
- Cover subtitle/body: `x=1.801 y=6.054 w=20.792 h=6.403`
- Default content title: `x=1.453 y=2.488 w=18.766 h=1.312`
- Default content body: `x=1.487 y=4.200 w=20.648 h=7.849`

## Use

- Default for readable business presentations and mixed content.
- Use light pages for sustained explanation and data.
- Many brand elements may be baked into background images; preserve the background rather than redrawing it.
