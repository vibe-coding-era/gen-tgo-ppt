# GTLC 大会 Design

Load this file when the user chooses `1. GTLC 大会`.

## Tone

- Formal, stage-ready, conference-grade.
- Prioritize authority, clarity, and high visual consistency.
- Prefer stronger opening/section rhythm than daily sharing.

## Template Defaults

- Default template: `light` for mixed business content.
- Use `dark` for keynote openings, major announcements, AI-native topics, and closing rhythm.
- Use `white` for dense reports, tables, or materials intended for printing.
- Do not default to `tgo-daily`; it is reserved for TGO日常活动分享 and other non-GTLC scenes unless the user explicitly requests it.

## Layout Priorities

- Cover: strong conference title, date, city, speaker, and event identity.
- Page 2: blank `嘉宾介绍` page unless guest information is provided.
- Agenda and chapter pages should make the event flow obvious.
- Content pages should feel suitable for large-screen viewing; avoid small, dense copy.
- Final page: `感谢聆听`.

## Style Fit

- Recommended visual styles: `1 Executive Board`, `3 Big Tech Keynote`, `4 AI Native`, `5 Data Intelligence`.
- Use `8 Creative Festival` only for creative/community sections, not as the whole-deck default.
