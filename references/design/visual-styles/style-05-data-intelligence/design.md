# Style 5: Data Intelligence

Load when the user chooses visual style `5`.

## Use For

Benchmarks, AI Coding evaluation, business analysis, operations analysis, and growth analysis.

## Keywords

Data, analysis, insight, growth, efficiency, metrics.

## Visual Direction

- Charts occupy at least 60% of data slides.
- Dashboard logic, rigorous hierarchy, clear metric grouping.
- Reference mood: Palantir, Snowflake, Databricks.

## TGO/GTLC Handling

- Prefer `white` or `light` templates for dense data.
- Keep chart geometry editable and legible.
- Use fewer metrics per slide when the page starts to wrap or crowd.
