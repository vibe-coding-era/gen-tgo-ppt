# Style 1: Executive Board

Load when the user chooses visual style `1`.

## Use For

CEO reports, board meetings, enterprise strategy, annual planning, and investment committees.

## Keywords

Professional, credible, restrained, rational, serious, strategic, decision-oriented.

## Visual Direction

- Deep blue, business gray, white.
- Large whitespace and data-first charts.
- Very few decorative elements.
- Reference mood: McKinsey, BCG, Bain, Fortune 500 board reporting.

## TGO/GTLC Handling

- Prefer `white` or `light` templates.
- Preserve top-right logo/footer.
- Prefer clean charts, decision tables, and concise executive claims.

## Content Expression

### Image Treatment

- Use only confirmed user photos, licensed/approved images, or approved AI-generated visuals; place one purposeful editorial or operational image beside the decision it supports.
- Favor restrained crops, cool-neutral color correction, and wide whitespace; use site, product, customer, or leadership imagery only when it evidences the claim.

### Diagram Language

- Use decision trees, option comparisons, operating-model layers, and concise arrows with thin blue-gray lines; make trade-offs and ownership legible.

### Data Expression

- Prefer clean editable bars, lines, waterfalls, tables, and variance callouts. Highlight only the metric that changes the decision.

### Avoid

- Avoid stock-photo collages, decorative AI imagery, saturated gradients, icon clouds, and visuals that do not affect a board decision.
