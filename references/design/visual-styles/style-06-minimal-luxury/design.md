# Style 6: Minimal Luxury

Load when the user chooses visual style `6`.

## Use For

Founder speeches, brand strategy, high-end forums, and company culture.

## Keywords

Premium, restrained, elegant, timeless, luxury.

## Visual Direction

- Black/white/gray.
- Extreme whitespace and very few elements.
- Typography-driven, editorial magazine feel.
- Reference mood: Apple, Hermes, LVMH.

## TGO/GTLC Handling

- Use blank or sparse layouts.
- Preserve brand chrome while reducing decoration.
- Prefer short, reflective statements over dense bullets.
