# Style 6: Minimal Luxury

Load when the user chooses visual style `6`.

## Use For

Founder speeches, brand strategy, high-end forums, and company culture.

## Keywords

Premium, restrained, elegant, timeless, luxury.

## Visual Direction

- Black/white/gray.
- Extreme whitespace and very few elements.
- Typography-driven, editorial magazine feel.
- Reference mood: Apple, Hermes, LVMH.

## TGO/GTLC Handling

- Use blank or sparse layouts.
- Preserve brand chrome while reducing decoration.
- Prefer short, reflective statements over dense bullets.

## Content Expression

### Image Treatment

- Use only confirmed user photos, licensed/approved images, or approved AI-generated visuals. Select one refined editorial image, material detail, portrait, or place when it carries the emotional weight of the statement.
- Use monochrome or subdued grading, considered crops, and extensive whitespace; an image is optional when typography says more.

### Diagram Language

- Use minimal sequences, fine-line comparisons, and quiet editorial annotations; reduce every diagram to its essential relationship.

### Data Expression

- Use one elegant number, a restrained line, or a concise comparison. Keep labels sparse and typography-led.

### Avoid

- Avoid image grids, glossy stock imagery, loud infographics, over-illustration, and filling empty space merely to make the slide feel busy.
