# Style 3: Big Tech Keynote

Load when the user chooses visual style `3`.

## Use For

AI conferences, technical conferences, product launches, and major announcements.

## Keywords

Futuristic, premium, breakthrough, innovation.

## Visual Direction

- Black or deep-blue backgrounds.
- Immersive visuals, full-screen imagery, huge numbers, huge titles.
- Minimal text and cinematic pacing.
- Reference mood: Apple Keynote, Tesla AI Day, Google I/O.

## TGO/GTLC Handling

- Prefer `dark` template for cover, chapter, and keynote pages.
- Use sparse copy and strong visual pacing.
- Move dense explanation to light/white support pages when needed.

## Content Expression

### Image Treatment

- Use only confirmed user photos, licensed/approved images, or approved AI-generated visuals. Prefer one cinematic full-bleed product, environment, or technological-detail image for a breakthrough message.
- Use dark, high-contrast grading, deliberate negative space, and minimal overlays; imagery should create scale or reveal the product, never act as wallpaper.

### Diagram Language

- Use oversized system paths, reveal-style sequences, and sparse luminous connectors; limit the visible structure to the idea being announced.

### Data Expression

- Feature one large performance number or a single high-contrast comparison; move detailed metrics to supporting light pages.

### Avoid

- Avoid small screenshots, crowded architecture maps, repeated neon effects, and decorative sci-fi imagery unrelated to the announcement.
