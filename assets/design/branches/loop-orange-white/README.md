# GTLC LOOP Orange White Branch

This folder stores the user-facing preview assets for the `GTLC-LOOP-OrangeWhite` branch style.

## Branch Identity

- Display name: `D｜LOOP Summit 橙白峰会风`
- Branch ID: `GTLC-LOOP-OrangeWhite`
- Role: a GTLC branch style for event-facing, sponsor-facing, and ecosystem summit materials
- Not a replacement for the GTLC `light`, `white`, or `dark` primary templates

## Assets

- `tgo-gtlc-style-branches.pdf`: 5-page customer selection guide.
- `tgo-gtlc-style-branches-contact-sheet.png`: overview image for the selection guide.
- `loop-2026-template-complete.pdf`: 14-page complete LOOP branch template preview.
- `loop-2026-template-complete-contact-sheet.png`: overview image for the 14-page branch template.
- `source/`: HTML sources used to generate the preview artifacts.

## Recommended Use

Use this branch when the customer needs a more active summit/event visual language:

- AI ecosystem summit
- sponsor benefits and partnership decks
- city chapter event promotion
- event agenda and speaker intro materials

Prefer the primary GTLC templates for formal reports, dense tables, board-style presentations, or print-heavy documents.
